# Wombat

Wombat: DevOps for non human beings

Wombat is a VM Lifecycle manager, it can creates virtual machines through libvirt, provision and configure using ansible, puppet or chef, has integration with cloud-init.

Components:

* Flask
* Swagger
* ReactJs

Plugins:

* wombat-ansible
* wombat-puppet
* wombat-chef
* wombat-salt
* wombat-vms
* wombat-ldap-auth
